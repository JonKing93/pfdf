Tutorials

.. admonition:: Download

  You can download the datasets and scripts used in these tutorials here: :doc:`Download Files <download>`.

    An introduction to Raster objects, their properties, and saving to file.

:doc:`Preprocessing <preprocess>`
    Use Raster objects to clean input datasets prior to assessment.

:doc:`Hazard Assessment <assessment>`
    Implement a standard hazard assessment.

:doc:`Parallel Basins <parallel>`
    Locate outlet basins using multiple CPUs.

:doc:`Monte Carlo Parameters <sweep>`
    Run a hazard assessment model using Monte Carlo sampling of the model's parameters.
