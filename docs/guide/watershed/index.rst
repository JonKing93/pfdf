Watershed Analyses
==================

This section introduces two modules that help analyze watershed datasets in preparation for hazard assessment modeling.

----

:doc:`Severity module <severity>`
    Functions to help classify BARC4-like soil burn severity (SBS).

:doc:`Watershed module <watershed>`
    Functions that analyze watersheds using output from a digital elevation model (DEM).
