data.usgs.tnm package
=====================

.. _pfdf.data.usgs.tnm:

.. py:module:: pfdf.data.usgs.tnm

Modules that load data products from the USGS's `The National Map (TNM) <https://apps.nationalmap.gov/downloader/>`_.

.. list-table::
    :header-rows: 1

    * - Module
      - Description
    * - :ref:`dem <pfdf.data.usgs.tnm.dem>`
      - Functions to load data from USGS digital elevation models (DEMs)
    * - :ref:`nhd <pfdf.data.usgs.tnm.nhd>`
      - Functions to load hydrologic unit (HU) datasets from the National Hydrologic Dataset (NHD)
    * - :ref:`api <pfdf.data.usgs.tnm.api>`
      - Functions supporting low-level TNM API calls
