models package
==============

.. _pfdf.models:

.. py:module:: pfdf.models

Modules that implement hazard assessment models.

.. list-table::
    :header-rows: 1

    * - Module
      - Description
    * - :ref:`staley2017 <pfdf.models.staley2017>`
      - Implements the debris-flow likelihood models of `Staley et al., 2017`_
    * - :ref:`gartner2014 <pfdf.models.gartner2014>`
      - Implements the potential sediment volume models of `Gartner et al., 2014`_
    * - :ref:`cannon2010 <pfdf.models.cannon2010>`
      - Implements the combined hazard classification scheme of `Cannon et al., 2010`_

.. _Staley et al., 2017: https://doi.org/10.1016/j.geomorph.2016.10.019
.. _Gartner et al., 2014: https://doi.org/10.1016/j.enggeo.2014.04.008
.. _Cannon et al., 2010: https://doi.org/10.1130/B26459.1
